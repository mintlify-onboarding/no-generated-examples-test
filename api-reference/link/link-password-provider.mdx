<RequestExample>

```bash Bash
curl --request POST \
     --url {{BASE_URL}}/v2/link/provider/password/{provider} \
     --header 'Accept: application/json' \
     --header 'Content-Type: application/json' \
     --header 'x-bazinga-link-token: <BAZINGA-LINK-TOKEN>' \
     --data '
{
     "username": "test",
     "password": "test"
}
'
```

const user = await client.Link.connectProvider(

  provider,

```

user = client.Link.password_provider(link_token, provider, username, password)
```
