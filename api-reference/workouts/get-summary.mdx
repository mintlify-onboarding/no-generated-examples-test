<RequestExample>

```bash Shell
curl --request GET \
     --url {{BASE_URL}}/v2/summary/workouts/{user_id} \
     --header 'Accept: application/json' \
     --header 'x-bazinga-api-key: <API_KEY>'
```

```python Python
from bazinga import Client

client = Client(
  api_key=<API_KEY>,
  environment="sandbox",
  region="us"
)

data = client.Workouts.get(
     user_id=*,
     start_date="2022-05-01",
     end_date="2022-06-01"
)
```

</RequestExample>
