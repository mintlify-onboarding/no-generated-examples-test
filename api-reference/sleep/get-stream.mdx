<RequestExample>

```bash Bash
curl --request GET \
    --url {{BASE_URL}}/v2/summary/sleep/{sleep_id}/stream \
    --header 'Accept: application/json' \
    --header 'x-bazinga-api-key: <API_KEY>'
```

```python Python
data = client.Sleep.getStream(sleep_id=*)
```

```js JavaScript
data = await client.Sleep.getStream(sleep_id=*)
```
